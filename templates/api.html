HTML→DAT変換 API

2ちゃんねるの HTML を dat 形式のテキストに変換します。
常識の範囲でご自由に使用してください。

（このサービスは Google App Engine の無料枠で作成されているため、無茶をするとすぐに使えなくなります。）

POST /api/dat
param html: 2chスレッドのHTML
response text: datに変換したテキスト
また、乱筆乱文ながらソースコードは https://github.com/wertrain/html-to-dat-py にあります。ご自由にご参照・ご利用ください。
